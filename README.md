# 🛡️ Cyber Threat Intelligence

An AI-powered Cyber Threat Intelligence (CTI) platform that automates cyber threat analysis using Machine Learning, Natural Language Processing, and Generative AI. The platform classifies cyber threats, extracts Indicators of Compromise (IOCs), identifies threat actors and malware, calculates threat severity, generates AI-driven remediation plans, and produces professional incident reports through a modern web interface.

---

## 🚀 Features

### 🤖 AI-Powered Threat Analysis
- Machine Learning-based threat classification
- AI-generated incident summaries
- AI-powered remediation recommendations
- Threat severity and risk scoring
- Intelligent threat categorization

### 🔍 IOC Extraction
Automatically detects and extracts:

- IPv4 Addresses
- Domains
- URLs
- SHA-256 Hashes
- CVE Identifiers
- Email Addresses

### 🧠 Natural Language Processing

- Named Entity Recognition (NER)
- Malware Detection
- Threat Actor Identification

- Threat Intelligence Parsing

### 📊 Dashboard & Analytics

- Threat Overview

### 📄 Report Generation

### 🔐 Authentication & Security
